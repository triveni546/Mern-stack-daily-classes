   variables:
   ------------
   -> variables are the named memory locations used to store data or values and it can change N number of times during execution
       str="JavaScript"
       str=10
    1. variable Declaration
    2. variable initialization
    3. variable utilization
    4. variable re-initialization
    5. variable re-Declaration

    in JS we define variables in 3days:
    -------------------------------------
    1. let  
        -> let variables acts as like a local scop/ block scop
        -> variable Declaration and initialization is same/ different line is posible
        ->re-initialization is posible but re-Declaration is not posible
        ->we can't hoist the let variables

    2. var
        ->var variables acts as like a global scop( because var present in windows object)
        -> variable Declaration and initialization is same/ different line is posible 
        ->re-initialization is posible and re-Declaration is not posible
        ->we can hoist the var variables
    3. const
        ->const variables acts as like a final  ( because once initialization values we can't modify data/values)
        -> variable Declaration and initialization is same/but different line is not posible 
        ->re-initialization and re-Declaration is not posible
        ->we can't hoist the const variables


    hoisting:
    ----------
    ->utilizing the variables before Declaration
    ->hoisting is posible
        1.variable Declaration
        2.variable initilization
        3.variable utilization
        4.variable Re-initilization
        5.variable Re-Declaration
